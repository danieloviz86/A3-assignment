About WEBD 162

WEBD 162 Beginning Web Development: HyperText Markup Language (HTML) and Cascading Style Sheets (CSS) This course introduces students to sound practices in creating a Website using HyperText Markup Language (HTML) and Cascading Style Sheets (CSS). Students create, validate and publish Websites to the Internet following Web standards. This course is intended for beginning Web design students and anyone interested in a career that requires skills and knowledge in Website construction. Associate Degree Credit and transfer to CSU.
